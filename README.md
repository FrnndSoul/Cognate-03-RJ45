# Cognate-03-RJ45

A lightweight multi-page website (HTML + CSS) for the RJ45 Cognate project.

## ✨ Features
- Static pages: Home (`index.html`), About (`about.html`), Contact (`contact.html`)
- Centralized styles in `/styles`
- Assets in `/resources`
